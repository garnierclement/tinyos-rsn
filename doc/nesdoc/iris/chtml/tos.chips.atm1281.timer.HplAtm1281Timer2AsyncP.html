Component: tos.chips.atm1281.timer.HplAtm1281Timer2AsyncP

module HplAtm1281Timer2AsyncP

HPL interface to Atmega1281 timer 2 in ASYNC mode. This is a specialised HPL component that assumes that timer 2 is used in ASYNC mode and includes some workarounds for some of the weirdnesses (delayed overflow interrupt) of that mode.
