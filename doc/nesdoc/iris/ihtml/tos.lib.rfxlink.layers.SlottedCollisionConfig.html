Interface: tos.lib.rfxlink.layers.SlottedCollisionConfig

interface SlottedCollisionConfig

Commands
command uint16_t getCollisionWindowLength(message_t *msg) Returns the size of the collision window for this received message.
command uint16_t getCollisionWindowStart(message_t *msg) Returns the start of the collision window for this received message, so transmit times in this range would be considered possible collisions.
command uint16_t getInitialDelay() This command should return the approximate transmit delay between setting an alarm, waiting for the fire event, calling send and obtaining the timestamp for the transmitted message.
command uint8_t getScheduleExponent() Must return a binary exponent so that the collision avoidance layer can assign slots in the range of [0, 1 << exponent) of size collision window.
command uint16_t getTransmitTime(message_t *msg) This command must return the time when the message was transmitted.
