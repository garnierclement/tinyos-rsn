Interface: tos.lib.rfxlink.util.RadioAlarm

interface RadioAlarm

Commands
command void cancel() Cancels the running alarm.
command tradio_size getNow() Returns the current time as measured by the radio stack.
command bool isFree() Returns TRUE if the alarm is free and ready to be used.
command void wait(tradio_size timeout) Waits till the specified timeout period expires.

Events
event void fired() This event is fired when the specified timeout period expires.

Commands - Details

cancel

command void cancel()

Cancels the running alarm. The alarm must be pending.

getNow

command tradio_size getNow()

Returns the current time as measured by the radio stack.

isFree

command bool isFree()

Returns TRUE if the alarm is free and ready to be used. Once the alarm is free, it cannot become nonfree in the same tasklet block. Note, if the alarm is currently set (even if for ourselves) then it is not free.

wait

command void wait(tradio_size timeout)

Waits till the specified timeout period expires. The alarm must be free.

Events - Details

fired

event void fired()

This event is fired when the specified timeout period expires.
