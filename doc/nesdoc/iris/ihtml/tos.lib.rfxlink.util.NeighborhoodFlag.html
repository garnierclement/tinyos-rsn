Interface: tos.lib.rfxlink.util.NeighborhoodFlag

interface NeighborhoodFlag

This interface provides one bit storage for each neighbor in a very fast and conveint way (without using shifts for example).

Commands
command void clear(uint8_t idx) Clears the flag for the given index.
command void clearAll() Clears the flag for all indices
command bool get(uint8_t idx) Returns the value of the flag for the given index
command void set(uint8_t idx) Sets the flag for the given index

Commands - Details

clear

command void clear(uint8_t idx)
