Interface: tos.lib.rfxlink.util.RadioChannel

interface RadioChannel

Commands
command uint8_t getChannel() Returns the currently selected channel.
command error_t setChannel(uint8_t channel) Sets the current channel.

Events
event void setChannelDone() This event is signaled exactly once for each sucessfully posted state setChannel command when it is completed.

Commands - Details

getChannel

command uint8_t getChannel()

Returns the currently selected channel.

setChannel

command error_t setChannel(uint8_t channel)

Sets the current channel. Returns EBUSY if the stack is unable to change the channel this time (some other operation is in progress), EALREADY if the selected channel is already set, SUCCESS otherwise.

Events - Details

setChannelDone

event void setChannelDone()

This event is signaled exactly once for each sucessfully posted state setChannel command when it is completed.
